# odin-recipes
Odin Recipes project from The Odin Project

This is a project from the "The Odin Project" Website where I'll be creating a recipe website. It will consist of a main index page which will have links to some recipes.

By the end of this project, I will have used the skills that I learned such as:
- "strong" for bold text
- "em" for italicized text
- variying headings
- paragraph tags
- either an absolute link and/or relative link to possible websites and web pages on the website I'll be creating
    - I'll also use the "target" and "rel" tag if I set a link to open in a new tab
- "img" tags for images as well
